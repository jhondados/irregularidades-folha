# 🏛️ Auditoria de Folha de Pagamento

> Auditoria inteligente de folha de pagamento pública. Detecta servidores fantasma, acúmulo ilegal de cargos e pagamentos irregulares.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/irregularidades-folha/

[![Vertical](https://img.shields.io/badge/Vertical-Governança-blue)]() [![IA](https://img.shields.io/badge/IA-ML-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**2.1M** Servidores · **8.400** Irregular. · **R$ 120M** Economia · **96%** Precisão

## 🧠 Tecnologias de IA
ML · Anomaly Detection · Entity Matching

## ✨ O que faz
- Detecção de servidor fantasma
- Acúmulo ilegal de cargos
- Cruzamento de bases
- Score de risco por vínculo

## 🗄️ Fonte de dados real
[Portal da Transparência](https://portaldatransparencia.gov.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Portal da Transparência
2. **Processamento com IA** — ML · Anomaly Detection · Entity Matching
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
